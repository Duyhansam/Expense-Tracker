# Expense Tracker

Ứng dụng quản lý chi tiêu cá nhân được xây dựng bằng ReactJS.

## Tính năng

* Thêm khoản chi tiêu
* Chỉnh sửa khoản chi tiêu
* Xóa khoản chi tiêu
* Tìm kiếm khoản chi tiêu
* Lọc theo tháng
* Lọc theo danh mục
* Thống kê tổng chi tiêu
* Thiết lập ngân sách tháng
* Lưu dữ liệu bằng Local Storage

## Công nghệ sử dụng

* ReactJS
* JavaScript (ES6+)
* CSS3
* Git & GitHub
* Vite

## Mục tiêu dự án

Dự án được xây dựng nhằm thực hành ReactJS, quản lý state, xử lý dữ liệu và xây dựng giao diện web tương tác.

## Cài đặt

```bash
git clone https://github.com/Duyhansam/Expense-Tracker.git
cd Expense-Tracker
npm install
npm run dev
```
